list-ping
===

list-ping is a shell program for bash that presents parallel execution ping command to multi targets.

## System
Bash 4.0 or later


## Installation
Use the command:
```bash

$ git clone https://github.com/gx3n-inue/list-ping

$ cd list-ping

$ vi list.txt
   ...
```

## Usage
$ ./list-ping.sh targetfile [ping count]"
